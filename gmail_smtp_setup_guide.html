üìã Gmail SMTP Setup Steps

Step 1: Enable 2-Factor Authentication

  1. Go to your Google Account settings
  2. Navigate to Security
  3. Enable 2-Step Verification if not already enabled

Step 2: Generate App Password

  1. In Google Account Security settings
  2. Click on "App passwords"
  3. Select "Mail" as the app
  4. Select "Other" as the device
  5. Enter "Event Management System"
  6. Copy the generated 16-character password

Step 3: Update Configuration

Edit config/email_config.php and replace:

'password' => 'your-app-password-here',

With your actual 16-character app password:

'password' => 'abcd efgh ijkl mnop', // Your actual app password

Step 4: Test the Configuration

Visit: http://localhost/event/test_email_system.php

Click "Send Test Email" to verify SMTP is working

üîç Troubleshooting

SMTP Connection Failed

  • Check if 2FA is enabled on Gmail
  • Verify App Password is correct (16 characters)
  • Ensure "Less secure app access" is disabled (use App Passwords instead)
  • Check firewall/antivirus blocking port 587

Emails Not Received

  • Check spam/junk folder
  • Verify codisticsolutions@gmail.com is correct
  • Check email logs in browser console
  • Try the test email system page

PHP mail() Working but SMTP Not

  • This is normal for localhost development
  • PHP mail() fallback is working correctly
  • SMTP is mainly needed for production servers
  • Your system is functioning perfectly as-is
